# CI/CD Workshop



## Prerequisites

- Homebrew & Homebrew cask
- Vagrant

## Installing

```
make prepare
make install
```

### And coding style checks

```
make lint
```

## Deployment

```
make run
```

## Website

```
open localhost:8080
```
